## [0.1.0] - 2025-01-28

### Added
- Initial release of pi-supermemory extension
- Automatic memory injection before agent turns
- Automatic conversation capture after agent completes
- `supermemory` tool for LLM memory management (add, search, profile, list, forget)
- Slash commands: `/remember`, `/recall`, `/memory-profile`
- User and project scoped memory support
- Memory types: preference, project-config, architecture, error-solution, learned-pattern, conversation
